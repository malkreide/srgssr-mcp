# CH-004 — OGD-CH Lizenz-Compliance: CC BY 4.0 Attribution
# Status: PARTIAL
# Reasoning: SRG SSR Public API ist nicht klassische OGD-CC-BY-4.0-Quelle (nicht-kommerziell, schriftliche Genehmigung für kommerzielle Nutzung). Die Lizenz-Lage ist im README dokumentiert (SRG SSR Terms of Use, kommerzielle Nutzung erfordert api@srgssr.ch). Aber: Tool-Antworten enthalten kein explizites "source"/"license"/"provenance"-Feld. README hat keine "Datenquellen & Lizenzen"-Tabelle wie im OGD-CH-Pass-Pattern.

## Modus: code_review (Source-Field in Tool-Returns)
$ grep -rnE '"source"|"license"|"provenance"|"attribution"' src/
(no hits — keine source/license/provenance Felder in Pydantic-Response-Models)
